Реализация консольного приложения для управления списком дел на Python.
Функционал:
- Добавить задачу
- Удалить задачу
- Отметить задачу выполненной
- Получить список задач
- Данные должны сохраняться между запусками в JSON
  
Требования:
- Python 3.9+,
- Понятные сообщения пользователю
- Работоспособность функционала
- Обработка ошибок

Функционал реализован в файле main.py, а данные сохраняются в tasks.json.
